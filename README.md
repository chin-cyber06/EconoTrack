# EconoTrack System

## Project Description
**EconoTrack** is a web-based business permit and registration tracking system created for the **Local Government Unit (LGU) of Butuan City**. The system aims to help administrators and registrars manage, monitor, and track business permit applications and renewals more efficiently.

This repository contains the project documentation, team style guide, and design resources used for the development of the EconoTrack prototype.

## How to View the Prototype

Since the project is currently in the prototyping stage, there is no source code or local server setup yet. To view and interact with the prototype:

1. Open the official Figma prototype link:**[EconoTrack Figma Prototype](https://www.figma.com/proto/IpoBqZKlKTUCUFbmWaW8ja/ECONOTRACK?node-id=287-357&t=wHi0I9HzyEhN0ZJY-0&scaling=contain&content-scaling=fixed&page-id=0%3A1)**.

2. Set the scaling option to **“Fit to Screen”** for better viewing.

3. Click anywhere on the screen to view the interactive hotspot areas.

4. Use the sidebar navigation to explore the different pages and dashboard components.
